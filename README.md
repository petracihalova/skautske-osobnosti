# Skautské osobnosti

Jednoduchá webová hra pro děti, která hravou formou představuje osobnosti spojené s českým a světovým skautingem.

## Jak hrát

Hra má dvě části:

- **Část 1 – Kvíz**: u každého jména vyber správnou fotografii a popis ze čtyř možností
- **Část 2 – Přiřazování**: přiřaď drag & drop všechny fotografie a popisy ke správným jménům

## Spuštění

Hra běží na GitHub Pages: **https://petracihalova.github.io/skautske-osobnosti/**

Nebo lokálně — stačí otevřít `index.html` v prohlížeči, hra nevyžaduje žádný server ani instalaci.
